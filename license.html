Linguistika

License

Linguistika is a dictionary that brings detailed grammar information for optimal language learning to mobile platforms. Russian dictionary data comes from OpenRussian, Tatoeba (released under CC BY 2.0 FR), Wiktionary, Dicipedia and Starling. This data is equivalent to the database published by OpenRussian (released under CC BY-SA), with minor structural changes, and is released under Creative Commons Attribution-ShareAlike 2.0 Generic (CC BY-SA 2.0). You can contact us for information on obtaining this data, but if access to the raw data is required for any purpose, using the original up-to-date database is recommended instead.
